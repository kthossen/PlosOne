# PlosOne

There are multiple models code in this repository. Models are given BILSTM,WBILSTM,LSTM,WLSTM ,LSTM-AT, WLSTM-AT, FLC,WFLC,BLC, WBLC .There are different files according to the timesteps 1 h to 64 hours. The published paper are found from the link https://journals.plos.org/plosone/article?id=10.1371/journal.pone.0314327
